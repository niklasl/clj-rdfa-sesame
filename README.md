# Sesame adapter for the Clojure RDFa 1.1 library

## Usage

## Development

To develop in tandem with clj-rdfa, use the `checkouts` support in Leiningen:

    $ mkdir checkouts && cd checkouts && ln -s ../../clj-rdfa
